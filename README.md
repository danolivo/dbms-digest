# DBMS Digest

**A weekly, ad-free, fact-checked roundup of PostgreSQL & wider DBMS internals.**

`PostgreSQL & DBMS Internals` · `Weekly — Mon → Sun` · `Signal over sales`

</div>

---

Each digest is a terse, fact-checked roundup of PostgreSQL and wider DBMS news,
research, and engineering for one **Monday → Sunday** week — ruthlessly filtered to keep the
signal and drop the marketing. Files live in [`digests/`](digests/), named by the Monday that
starts the covered week. New digests are generated by the `weekly-dbms-digest` skill; see
[`CLAUDE.md`](CLAUDE.md) for the auto-run routine.

> **Think you can make the digest sharper?** The generator prompt lives in
> [`.claude/skills/weekly-dbms-digest/SKILL.md`](.claude/skills/weekly-dbms-digest/SKILL.md).
> PRs that improve the curation, filtering, or fact-checking — or add a high-signal source to
> [`sources.md`](.claude/skills/weekly-dbms-digest/references/sources.md) — are very welcome.
